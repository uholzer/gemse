A [ X ] , f ( X ) (Polynom)
Sei A ein kommutativer Ring. Betrachte eine unendliche zyklische Gruppe erzeugt durch X. Sei S = { X r : r ∈ ℕ ∪ { 0 } } . Dann ist A [ X ] = { f : S → A : ∣ { s ∈ S : f ( s ) ≠ 0 } ∣ < ∞ } die Menge von Polynomen. Wir schreiben a X n ∈ A [ X ] für diejenige Funktion f : S → A mit f ( X n ) = a ∧ ∀ s ∈ S : s ≠ X n ⇒ f ( s ) = 0 . Elemente aus A [ X ] werden als f ( X ) geschrieben.
