This list includes all movement commands from the edit mode. They are not documented here. You use them to move around, that is, to change the selection. However, such a command only has an effect if it moves to a sibling of the current position.

Move the other end of the selection

After issuing this command, the next movement commands will move the other end of the selection.

Cancel

Closes the visual mode and drops the selection
