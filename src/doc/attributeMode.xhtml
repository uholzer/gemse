Attribute Mode

Commands

CommandtitleFunction name

Move to next attribute

Moves the cursor to the next attribute.

Move to previous attribute

Moves the cursor to the previous attribute.

Delete attribute

Deletes an attribute.

Inserts an attribute using the default namespace

Use this command to insert an attribute that is in the default namespace (which applies to all MathML attributes). After this command, write the name of the attribute, hit enter, write the value and hit enter again.

Inserts an attribute using the given namespace

Use this command to insert an attribute that is not in the default namespace. After this command, write the namespace of the attribute, hit enter, write the name, hit enter, write the value and hit enter a third time.

Change Value

Changes the value of the attribute selected by the cursor. Simply write the new value behind the command and hit enter.

Delete all

Deletes all attributes

Set attributes according to dictionary

On an mo element, you can use &enter; to set the attributes according to the dictionary entry for this operator, as shown in the dictionary view. If there is more than one entry, the one is chosen wich has the same form, as given in the form attribute of the element, or, if not present, as deduced automatically. If there are still multiple choices, you can enter the disambiguation string after the command, before hitting enter.

Exit

Terminates attribute mode.
