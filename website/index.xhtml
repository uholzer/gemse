Gemse, the General Math Structure Editor

The project Gemse has been started in the need for an editor for mathematical formulas that is fast and can be controled by keyboard only.

Actually, the first approach to this problem, "Gemse Approach 1", is an editor for MathML. It is currently in development.

If you are looking for an editor that catches the semantics of a formula, you should consider Sentido. Also take a look at MathEdit.
