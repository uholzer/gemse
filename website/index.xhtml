Gemse, the General Math Structure Editor

Gemse 1.0.8 has been published!

The project Gemse has been started in the need for an editor for mathematical formulas that is fast and can be controlled by keyboard only.

Approach 1

Actually, the first approach to this problem, "Gemse Approach 1", is an editor for Presentation MathML. It also has some limited abilities to edit Content MathML. It is currently in development.

Other solutions

You can find other solutions on the software list of the W3C. Additionally there is Sentido which is used for example in the editor of SWIM (test it).
